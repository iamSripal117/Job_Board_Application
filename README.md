# JobSphere - Modern Job Board Application

## Overview

JobSphere is a modern job board application built using HTML, CSS, and JavaScript. The platform allows users to browse jobs, save jobs, apply for jobs, upload resumes, and manage their job search through a personalized dashboard.

---

## Features

### User Authentication

* User Registration
* User Login
* Session Management using Local Storage
* Logout Functionality
* Protected Dashboard Access

### Job Management

* Browse Available Jobs
* Search Jobs
* Filter Jobs by Category
* View Job Details
* Save Jobs
* Apply for Jobs
* Recently Viewed Jobs

### User Dashboard

* Profile Overview
* Saved Jobs Management
* Application History
* Recently Viewed Jobs
* Resume Upload
* Profile Completion Tracking
* Personalized Recommendations

### Responsive Design

* Mobile Friendly Layout
* Tablet Support
* Desktop Optimization
* Modern UI Design

### Dark Mode

* Toggle Between Light and Dark Themes
* Persistent Theme Preference

### Data Persistence

* Local Storage Based User Data
* User-Specific Saved Jobs
* User-Specific Applications
* User-Specific Resume Storage
